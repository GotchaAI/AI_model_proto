# Get Started

python 3.10.16 버전을 기준으로 작성되었습니다.

`pip install -r requirements.txt (또는 pip3 install -r requirements.txt)`

를 입력하여 필요한 패키지를 설치합니다.

`python app.py (또는 python3 app.py)`

를 입력하여 서버를 실행합니다.
